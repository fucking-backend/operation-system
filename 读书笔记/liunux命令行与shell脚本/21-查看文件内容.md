
## file 查看文件类型


```bash
# 查看文件类型
file errors.log



```


## 查看整个文件

### cat 查看所有内容

```bash
# 查看文件内容
cat  errors.log

# 查看，并显示行号
cat -n errors.log


# 查看，并显示行号，只给有文本的行添加行号
cat -b errors.log
```


### more 分页查看

```bash
more errors.log
```


### less 分页查看

```bash
less errors.log
```

##  查看部分文件

### tail

```bash
tail errors.log
```


### head

```bash
head errors.log
```
